Security Overview

CallPilot AI applies technical and operational controls for SaaS security baseline readiness.

Current Controls

  • OTP-based login with attempt limits.
  • Session cookies with HttpOnly and SameSite protections.
  • Webhook signature validation for Twilio and Stripe.
  • Rate limiting for sensitive API endpoints.
  • Audit logging for critical admin actions.

Responsible Disclosure

Please report vulnerabilities to security@callpilot.ai.
